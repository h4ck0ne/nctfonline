# nctfonline

南邮ctf（网络攻防）在线答题系统。利用闲暇时间开发，将代码共享于此，方便其他学校的想办ctf比赛孩子使用。
数据库文件是nctfonline.sql 。

 
## tips
管理员账户为ctfadmin ctfadmin 
登陆后自动跳转到后台管理界面
